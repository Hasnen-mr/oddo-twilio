# Oddo Twilio (Odoo Apps Store)

This repository contains **one Odoo module** at the root, as required by the Odoo Apps Store.

## Structure

```
oddo-twilio/           (repo root – when you clone)
  odoo_twilio/         (module folder – one folder at root)
    __manifest__.py
    __init__.py
    models/
    views/
    ...
```

- **Repository URL (18.0):** `ssh://git@github.com/Hasnen-mr/oddo-twilio.git#18.0`
- **Module name:** `odoo_twilio` (Twilio Connector – Calling, SMS and Call Logs)

Install by adding this repo to your Odoo addons path and installing the app, or install from the Odoo Apps Store.

See `odoo_twilio/README.md` for module usage and configuration.
