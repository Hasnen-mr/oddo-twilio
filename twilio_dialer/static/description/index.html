How It Works

Set up in under 2 minutes. No tutorials, no learning curve.

1
Install the Odoo Module Copy the twilio_dialer folder to your Odoo addons path, update the Apps list, and install.
2
Configure Twilio Credentials Go to Twilio → Configuration in Odoo. Enter your Account SID, Auth Token, and Twilio phone number.
3
Start Calling & Texting Open any Odoo contact or lead and click the call or SMS button. All activity is automatically logged.
4
View History & Reports Browse the full call and SMS history in the Twilio menu. Filter, export, and track all communications.
